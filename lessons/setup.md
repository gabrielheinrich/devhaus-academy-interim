## SSH Key

In order to be able to access your private server, we need to generate an ssh key.
An ssh key comes with two parts a private and a public one. Make sure to never ever share the private part with anyone.

[How to generate an ssh key](https://docs.github.com/en/github/authenticating-to-github/generating-a-new-ssh-key-and-adding-it-to-the-ssh-agent)
